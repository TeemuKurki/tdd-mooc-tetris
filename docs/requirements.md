# Tetris requirements

1.  Spawn block on top of screen
2.  Block falls at constant speed
3.  Block can be move left and right while falling
4.  Block can be rotated
5.  Block falling speed increase while pressing down
6.  Block instantaniously falls bottom when pressing spacebar
7.  When block is at bottom it is stuck there
8.  When every square in row has block, blocks on that row disappear
9.  Stucked blocks at botto becomes "new floor"
10. Get point while blocks are falling
11. Level up when x number of rows has disappeared
12. Level up increases block falling speed
